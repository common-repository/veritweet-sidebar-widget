Plugin contains sidebar widget, that allows to view Veritweet channel posts.

== Description ==

Plugin displays latest [Veritweet.com](http://www.veritweet.com/ "Veritweet") channel posts in WordPress website sidebar. User has to enter channel ID, channel name and number of posts that should be shown. 

Posts only from verified Veritweet accounts are shown.

Tu fully use the plugin, user has to vreate Veritweet account first and verify it. 

== Installation ==

1. Download the plugin
2. Upload it in your plugin directory (.../wp-content/plugins/)
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Under the 'Appearance' submenu 'Widgets' place Veritweet widget in your sidebar
5. Enter Veritweet channel name, Channel ID number (can be found in channel hyperlink) and number of posts to be viewed (up to 20)
6. Enjoy


== Frequently Asked Questions ==

= Why am I getting a message 'Channel not verified. Please verify to view posts!'? =

Posts only from verified channels are displayed. You have to verify your channel on Veritweet.com. If you are not the owner of channel you want to display, then there are no other options for displaying the posts (channel owner hasn`t been verified)

= Where do I find channel ID? =

Open Veritweet channel. Channel ID in a form of numbers can be found in the hyperlink of particular channel.
